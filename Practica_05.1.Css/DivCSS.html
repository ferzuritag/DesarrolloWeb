este es un ejemplo demostrativo de texto

Este ejemplo esta enfocado en demostrar la decoracion de texto, transformacion de texto, alineacion de texto, identacion de texto y altura de linea.

Note que el parametro line-height provoca que el espacio entre las lineas sea mas grande de lo normal guiado por las etiquetas DIV. Los Elementos DIV son textos alineado a la derecha, en azul, subrayados y la primer letra de cada palabra es una Mayuscula.

Note tambien que el texto en el Elemento H1 esta todo en letras mayusculas aun cuando todo el texto fue tecleado en letras minisculas.
